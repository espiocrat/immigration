# 出入国行政の透明性ダッシュボード

[한국어](README.md) · [English](README.en.md) · **日本語**

法務部（韓国）の出入国・外国人政策統計を加工し、全国17か所の出入国・外国人官署の**管轄区域**とビザの**処理予想日数**を単一HTMLのダッシュボードで可視化するプロジェクトです。韓国に滞在する外国人が、自分の地域の担当事務所、書類の受付窓口（本庁・出張所）、予想処理期間を一画面で確認できるようにすることを目的とします。

ビザの延長・変更を申請した後の処理期間は公開されておらず、申請者は進捗を把握しづらいのが実情です。本ダッシュボードは公開統計のみからその期間を推定し、管轄・受付窓口の情報を併せて提供します。

## 機能

- **処理日数の予測ツール** — 事務所・ビザ・申請月を入力 → 予想日数、完了予定日（ETA）、95%信頼区間
- **管轄区域マップ（17事務所）** — 251の市・郡・区を公式の管轄に従って事務所領域に統合し、予想日数で色分け。事務所にカーソルを合わせると、管轄の市・郡・区、受付窓口（本庁・出張所）、処理統計を表示。地図が読み取りにくい利用者向けに格子状の**タイル表示**も用意
- **地域の流入・流出** — 道・市単位の月間純増減（新規流入−流出）のコロプレス図
- **処理件数の推移** — 実測52か月（2022.01–2026.04）＋ OLS 6か月予測
- **ビザ別分析 ・ ビザ構成 ・ 季節性 ・ 事務所ランキング**
- 韓国語／英語／日本語 ・ ダーク／ライト（システム連動） ・ レスポンシブ（モバイル／タブレット／デスクトップ）

## データ

| 項目 | 出典 | 基準 |

データはすべて法務部の公開資料（公共ヌリ 第1類型）です。

## 管轄マッピング

17事務所の管轄は、法務部が公開する**出入国管理事務所の現況**（公式の管轄区域）に基づいて算定します。

1. 「登録外国人 市郡区別 居住現況」（2026.04、255行）から市・郡・区別の登録外国人を抽出
2. 公式の管轄表に従って各市・郡・区を17事務所へ割り当て。出張所は母事務所へまとめるが、**越境的な割り当て**は公式どおりに反映 —— 例：慶北の慶州→蔚山、慶南の金海・密陽・梁山→釜山、江原の鉄原→楊州、京畿の金浦→仁川、京畿の安養・光明→ソウル
3. 割り当て後に**未マッピング0件・合計一致**（1,623,256）を検証
4. 市・郡・区のポリゴンを事務所単位でディゾルブ → メルカトル投影 → 軽量SVGパスとして事前ベイク

この手順により、ウェブ調査に依存していた従来マッピングの越境管轄の誤り約10件を公式資料で修正しました。事務所のツールチップには本庁・出張所ごとの受付管轄も併記し、外国人がどこで書類を受け付け、どの事務所が処理するのかを区別できるようにしています。

## 予測モデル

月別の処理件数をOLSトレンド＋季節分解で学習しています。R² 0.958、MAE 5.1%、RMSE 16,048件、トレンド +2,802件/月、繁忙期は8月（+44,948件）。

事務所・ビザ別の予想日数：

```
日数     = 基準日数 × 負荷指数^0.5
負荷指数 = （管轄の登録外国人 ÷ 処理能力等級）÷ 全国平均
処理能力等級 = 庁 1.30 / 事務所 1.00 / 出張所 0.55   （定員は非公開のため等級で近似）
```

画面では実測（REAL）と推定（EST）を区別して表示します。モデルの仮定は事務所の処理能力等級の一つだけで、他の入力値（管轄の登録外国人・管轄区域・流入/流出）は出典のある実測です。定員や処理実績が公開されれば、この仮定も実測に置き換えられます。

## 技術構成

- 単一のHTMLファイル。ビルドもサーバーも不要で、ブラウザで開くだけで動作
- 地図：自己完結型のSVG。市・郡・区の境界を事務所領域に統合して投影（メルカトル）し、d3/topojson/外部fetchは使用しない。管轄マップと流入・流出マップは同一のviewBox／座標系を共有
- フォント：韓国語（Nanum Gothic）・日本語（Noto Sans JP）を使用する文字だけにサブセット → woff2 → base64で埋め込み（合計約248KB）。フォントCDN非依存、オフライン動作。ラテン文字・数字はシステムフォント
- 外部依存：チャートライブラリ（Chart.js）1つのみ

## 補足

- 「滞在外国人」（約287万）と「登録外国人」（約162万）は別の母集団です。すべての数値にどの人口かを併記しています。
- 非正規滞在の外国人への差別を招きうる不法滞在率などの指標は含めていません。
- 本ダッシュボードは推定値に基づくもので、法的効力はありません。公式統計ではなく、法務部とは無関係の独立した学術プロジェクトです。

## 実行 / 公開（GitHub Pages）

`index.html` をリポジトリのルートに置き、**Settings → Pages → Source** を `main` ブランチのルートに設定すると、`https://<ユーザー>.github.io/<リポジトリ>/` で公開されます。

```
immigration/
├── index.html        # ダッシュボード（単一ファイル、フォント・地図を内蔵）
├── README.md         # 한국어
├── README.en.md      # English
└── README.ja.md      # 日本語
```

データ更新時：新しい市郡区CSVで管轄を再集計 → 未マッピング0・合計一致を再検証 → 地図を再ベイク。画面の文字が変わった場合はフォントのサブセットを作り直します。
